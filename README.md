# L2C-FL: A Privacy-Preserving Federated Hybrid Transformer Framework with Iterative Artificial Protozoa Optimization for Multi-Modal Lung Cancer Classification
Lung cancer is considered as the leading causes of cancer-related mortality worldwide that accounts for the significant number of deaths due to delayed diagnosis and the complexity of identifying malignant pulmonary nodules in medical images. Still, the development of robust deep learning models is limited with lack of access to large-scale medical datasets, privacy concerns, heterogeneous data distributions across healthcare institutions and high computational requirements. Federated Learning (FL) has designed for enabling collaborative model training without sharing sensitive patient data. Still, existing FL-based approaches has challenges like inefficient feature extraction, unstable client participation, communication overhead and suboptimal convergence. To address the limitations, this research proposed the Federated Learning-based Lung Cancer Classification model with preprocessing, adaptive client selection, sparse attention mechanisms and loss function optimization. Initially, the input images are pre-processing using Contrast Limited Adaptive Histogram Equalization (CLAHE), normalization and image resizing to enhance image quality and standardize input representations. In this, the local model is designed using the Transformer model with Parallel Sparse Attention, Hybrid Inception Simple Recurrent Unit (INC-SRU) model. Besides, to improve federated aggregation reliability, the Dynamic Edge Thresholding-based Client Selection strategy is employed to identify and select stable client updates based on gradient variance statistics. Then, the model loss function optimization is employed using the proposed Iterative Artificial Protozoa Optimization (IAPO) algorithm to enhance convergence and avoid local optima.
